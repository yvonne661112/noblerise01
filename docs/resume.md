## 任務狀態：✅ 已完成（2026-03-27）

## 任務目標

對 `blog/` 下所有 34 篇 markdown 文章進行排版優化，規則如下：

1. **結構**：確認有 frontmatter（title / description / date / tags）
2. **標題**：h2 作為主要段落，h3 作為子項，不使用 h4 以下
3. **清單**：有序清單用於步驟流程，無序清單用於並列項目
4. **強調**：重要名詞或金額用 **粗體**，警示用 `>` 引言區塊
5. **表格**：數字比較內容改為 Markdown 表格格式
6. **空行與斷行**：
   - 每個 h2 段落前空一行，內文段落間空一行
   - 連續兩個以上的空白行，合併為一個空白行
   - 句子中間出現不合理的換行（同一語意未結束即斷行），合併回同一段落
7. 不修改任何中文文字內容
