根據[道安總動員](https://roadsafety.tw/)統計2023年台灣車禍事故高達40萬2,926件，因車禍受傷人數達53萬9,535，死亡人數達3,023人。開車騎車不宜搶快，剛畢業時騎機車上下班，常在通勤路上看到車禍，時常幫忙報警或叫救護車處理，自己也因為別人下車沒有看後方來車直接開車門，讓我摔了2次車，還好命大，後方沒有來車，手腳挫傷一切平安。

如果汽車遇到車禍該怎麼做？正確處理6步驟，減少損失！

#### **步驟1：車禍事故現場處理**

1. 汽車在一般道路發生碰撞車禍，先打閃黃燈下車後，應於車後豎立三角錐警告牌，避免二次車禍。
    
2. 盡速打110報警，如果有傷患告知事故地點並通報有人受傷，警方會主動通報119救護車到現場，碰撞現場一定要確保傷患在安全位置。
    
3. 拍攝事故現場照和碰撞點的照片，拍攝技巧由遠方看到路線和撞擊點、跡落物、煞車痕，近距離拍車體損害照片，記得要拍到車牌，檢查自己的行車紀錄器，將事故當下先截錄並儲存，避免紀錄被洗掉，可做為事故過程的證據。觀察四周或附近店家是否有監視錄影器，並記錄十字路口的紅綠燈秒數。
    
4. 警察做筆錄時，詢問事故經過和行使路線、號誌、車速、撞擊位置等狀況，務必確認筆錄和現場圖沒有問題才簽名，接下來警察會開始拍現場照和事故車損害狀況，確保是這次事故造成的損害，做完筆錄後會拿到『道路交通事故當事人登記聯單』。
    
5. 有投保產險公司的道路救援服務，可以打道路救援電話申請拖吊服務，建議拖回原廠維修或自己平日的修車廠。拖吊車服務是依據行車距離收費
    
6. 修車廠進行車損估價時，可以請修車廠服務人員或通知產險公司辦理出險。出險流程需要駕駛人駕照和車子行照、事故當事人登記聯單、填寫理賠申請文件，車輛維修必須等產險公司理賠人員勘車估價，確認損害部位和維修金額後再進行維修。

#### **步驟2：申請事故現場圖、現場照片、初判表**

申請事故現場圖、現場照片、初判表，就是讓自己了解此次車禍的肇責比例，車禍處理調解和上法院第一件事都是先談肇責比例，肇責比例高低，會影響賠償對方的金額比例或自己受傷請求損害的比例，以下為申請流程：

- 道路交通事故案件需當場通知警方到場處理，案件有人員死亡或受傷者，才能在內政部警政署『[道路交通事故資料申請系統](https://tm2.npa.gov.tw/NM105-505ClientRWD2/TM02A01Q_01.jsp)』申請服務，無人傷亡之案件，請洽原受理警察單位。
    
- 道路交通事故案件當事人或利害關係人，得於下列期間登入警政署『[道路交通事故資料申請系統](https://tm2.npa.gov.tw/NM105-505ClientRWD2/TM02A01Q_01.jsp)』申請提供相關資料：
    

```
事故七日後得申請提供事故現場圖、現場照片。
```

```
事故三十日後得申請提供道路交通事故『初步分析研判表』。
```

若對交通事故『初步分析研判表』的分析結果不服，可以申請『交控系統錄影資訊調閱』，找出對自己有利的證據和道路交通規則法條，另花3,000元事故日起6個月內在監理服務所下載**「**[**車禍鑑定**](https://www.mvdis.gov.tw/m3-emv-vil/cac/download#gsc.tab=0)**」****申請表**，列印填寫後，親自前往或郵寄至事故發生地點[管轄之鑑定會](https://www.mvdis.gov.tw/m3-emv-vil/cac/contact)申請。

#### **步驟3：釐清車禍法律責任**

接到小陳慌張的電話：「媽媽騎機車直行行駛路上被後方來車撞上摔車受傷，後方來車是汽車，車上駕駛沒有受傷，媽媽無照駕駛，這樣可以申請強制險理賠嗎？」此案法律責任如何區分？

- **行政責任：**陳媽媽無照駕駛，依照『[道路交通管理處罰條例第21條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=K0040012&flno=21)』規定，需要被裁罰6,000-24,000行政罰款。
    
- **刑事責任：**由於後方來車未保持安全距離撞上前車，違反『[道路交通安全規則第九十四條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=K0040013&flno=94)』造成陳媽媽摔車受傷，陳媽媽可以依據[刑法284條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=C0000001&flno=284)於事故日起半年內到警察局提出過失傷害告訴，可以用口頭做筆錄方式為之。 
    
- **民事責任：**受傷期間的損害譬如醫療費用、少領的薪資所得、精神撫慰金等，陳媽媽可以於事故日起2年內依據[民法184條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=B0000001&flno=184)、[民法195條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=B0000001&flno=195)向法院提出行使民事損害賠償請求權。

#### **步驟4：和解不成可申請調解**

經過治療後，雙方對於肇責比例都有共識，就可以請產險公司和陳媽媽談和解，和解書的內容除了賠償金額之外，務必要求對方放棄民、刑事之追訴權；如果理賠金額沒有共識，可自行向事故發生地的「鄉鎮市區公所調解委員會」之「調解聲請」，以[台北市政府市民服務大平台參考調解書和範例](ttps://service.gov.taipei/Case/ApplyWay/201907120127)，可以書面和線上申請調解，申請請依當地調解委員會規定。

雙方收到調解委員會通知，調解會在鄉鎮市區公所調解委員會進行，通常調解程序不會公開，會議中主席會先詢問雙方肇責比例、賠償項目和金額，雙方設定的賠償條件、金額底線等，了解雙方想法後再進行調解，調解結果分為「調解成立」和「調解不成立」，調解成立之後可送至法院公證，其效力等於法院判決。如果調解不成立又快滿6個月時間，陳媽媽就可以直接到附近的警局向警員說要提告過失傷害，一場漫長的法律流程就此展開！

#### **步驟5：法院流程**

陳媽媽在提告後，被告會收到地檢署傳票開庭，檢察官會主持偵查庭，偵查流程不公開，一般只有本人和律師可以進去偵查庭，檢察官會詢問:

- 雙方對肇責有沒有異議
    
- 雙方有沒有和解的意願
    

如果雙方有和解意願，檢察官就會發回調解委員會進行調解。

如果調解成功，依據[刑法第287條](https://law.moj.gov.tw/LawClass/LawSingleRela.aspx?PCODE=C0000001&FLNO=287&ty=L)因過失傷害為告訴乃論之罪，陳媽媽可以撤銷過失傷害告訴。

如果調解不成，偵查結束後，檢察官會依據被告的過失責任起訴或聲請簡易判決，交給法官接手處理。

文件轉送刑事法院後，在刑事法院間雙方如沒有調解成功，受傷期間的損害如醫療費用、少領的薪資所得、精神撫慰金等，陳媽媽可以在事故日起2年內依據[民法184條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=B0000001&flno=184)、[民法195條](https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=B0000001&flno=195)向法院提出行使損害賠償請求權，可以採用**刑事附帶民事訴訟**，讓受害人可以直接在刑事訴訟程序中向加害人請求民事賠償，不用繳民事裁判費，被告就會被法官判刑，詳細可以參考過失傷害判決，如果在官司上有任何問題可以詢問專業律師。

#### **步驟6：保險理賠需知**

#### **強制險**

強制險理賠必須在兩造事故的前提之下，譬如說車撞人、車撞車的情況下，理賠車主以外第三人，如果車主自行開車撞牆、安全島、騎車撞到狗摔車，都是屬於單一事故，就不在強制險的理賠範圍內，建議駕駛人要投保個人意外險、強制附加駕駛人傷害險、第三人責任險附加駕駛人傷害險。

如果是行人走路被車撞、車子和機車碰撞造成有人受傷，受傷的人都可以向對方的產險公司申請強制險理賠，強制險採無過失主義，只要被車子碰撞造成受傷，不論肇責都可以申請理賠。

![](/images/強制責任險給付內容.webp)

#### **第三責任險、超額責任險、失能增額險**

強制險可以申請基本醫療費用，如果有自費醫療費用和請假在家休養薪資損失、車子損害、受傷期間精神折磨可以請求精神賠償等，這些都需要第三人責任險啟動，如果是重大車禍事故，對方身故或重殘或者開千萬跑車，建議另外增加超額責任險至少1,000萬以上還有失能增額險，提醒留意車險的續保時間，一定要正常續保，不能斷保，曾經遇過朋友斷保慘賠7位數，避免一場車禍造成財務土石流甚至背上負債人生。

#### **張惠芳CFP小結**

遇到車禍該怎麼做？車禍處理六步驟SOP：

1. 現場處理與保護證據：打警示燈、放三角錐、報警並拍攝事故證據
    
2. 申請現場圖和照片、初判表，確定肇責比例
    
3. 釐清法律責任，了解行政、民事及刑事責任
    
4. 優先申請調解，避免訴訟
    
5. 若進入法院程序，檢察官會主持偵查庭及發回調解程序，若無法調解成功，建議採刑事附帶民事訴訟。
    
6. 保險理賠知識，強制險理賠基本醫療費用，第三責任險涵蓋自費醫療費、車損、薪資賠償及精神賠償，超額責任險與失能增額險則可應對重大的財務和損害賠償風險。
    

車禍處理的核心在於適時蒐集證據、釐清責任、尋求和解，必要時再透過法律途徑維護自身權益，並在保險理賠、法律責任和後續賠償方面獲得完整的保障，確保將自身的財務風險降到最低，減少損失！
